Created a simple todo application as a coding challenge

[Link](https://ancient-waters-49479.herokuapp.com/todo)
## To run Locally
 Git Clone this repo  
 cd into folder  
`npm install`  
`npm start`    
Add .env file with API URL and variable name `REACT_APP_API_BASE_URL`
